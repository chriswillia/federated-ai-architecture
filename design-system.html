Honest tensions

Where the framework is opinionated.

Trade-offs we acknowledge up front, with primary-source links to how each one is currently resolved.

Tension · Citizen builders in regulated environments

The framework permits citizen-built agents to reach PII through governed connectors. In regulated industries, that requires approval workflows the platform supports but does not enforce by default.

Tension · Multi-cloud identity

Using BYO identity providers across clouds (Entra + Okta + AWS IAM) requires federation testing the framework outlines but does not provide turnkey.
